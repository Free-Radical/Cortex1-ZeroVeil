# Security Policy

## Reporting a Vulnerability

Please report security issues privately.

- Email: Saqib.Khan@Me.com
- Subject: `SECURITY: Cortex1-ZeroVeil`

Include:
- A clear description of the issue and impact
- Steps to reproduce
- Any proposed fix/mitigation

## Disclosure

We aim to acknowledge reports within 7 days and provide a remediation plan as quickly as feasible.

Please do not file public GitHub issues for vulnerabilities.
